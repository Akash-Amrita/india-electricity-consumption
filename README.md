# ⚡ India Electricity Consumption Analysis (2019-2020)

## 📌 Project Overview
Analysis of electricity consumption patterns across Indian states 
from January 2019 to December 2020, covering the COVID-19 pandemic period.

## 🛠️ Tools Used
- **MySQL** — Database storage and SQL queries
- **Tableau** — Data visualization and dashboard
- **Python Flask** — Web application

## 📊 Key Features
- 12 interactive Tableau visualizations
- Regional consumption analysis (NR, SR, WR, ER, NER)
- COVID-19 lockdown impact analysis
- Before/During/After lockdown comparison
- Flask web app with embedded Tableau dashboard

## 🔍 Key Findings
- Western Region had highest consumption in 2019 (3,85,280 MU)
- COVID-19 lockdown caused significant drop in electricity usage
- Maharashtra was the top consuming state
- Most states recovered to near-normal levels by end of 2020
