About This Project

This project analyzes electricity consumption patterns across Indian states from January 2019 to December 2020, covering the COVID-19 pandemic period.

Key Findings

⚡ Western Region had highest consumption in 2019 (3,85,280 MU)

📉 COVID-19 lockdown caused a significant drop in usage

📈 Most states recovered by end of 2020

🏭 Maharashtra was the top consuming state
